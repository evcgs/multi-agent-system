# Global Awareness 全局感知技能

所有Agent的基础能力，确保具备主动信息收集和全局视角，避免信息片面。

---

## 核心功能

### 1. 会话初始化扫描
**触发时机**：每次会话开始时自动执行
**功能**：
- 读取 MEMORY.md 和最近2天的记忆文件
- 扫描工作区所有项目和Agent
- 生成项目状态全景图
- 识别待推进任务和阻塞点

### 2. 信息完整性校验
**触发时机**：回答任何项目相关问题前自动执行
**校验项**：
- 是否扫描了所有相关目录的文件？
- 是否关联了分散的信息点？
- 是否有遗漏的配置、规范、代码？
- 结论是否有足够的文件支撑？
**处理规则**：任何一项不满足则先完成信息收集再回答，禁止给出片面结论。

### 3. 项目全景报告生成
**调用方式**：`生成项目全景报告`
**输出**：包含Agent状态、配置状态、任务完成状态、整体进度的标准化报告。

---

## 使用方式

### 作为Agent基础能力
所有Agent默认继承此能力，无需显式调用，会自动在会话初始化和回答问题前执行。

### 手动调用
```
"生成当前项目状态报告"
"扫描所有Agent状态"
"检查项目完成度"
```

---

## 能力要求
所有Agent必须具备此能力，在Agent初始化验证时必须通过以下测试：
> 问题：你知道当前有哪些已完成的Agent？项目整体进展如何？
> 预期回答：能准确说出已完成的Agent名称、完成度、待推进任务，证明具备全局感知能力。
