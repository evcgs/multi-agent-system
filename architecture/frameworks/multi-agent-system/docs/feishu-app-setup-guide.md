# 飞书应用创建指南

本指南指导你如何在飞书开放平台创建多个飞书应用。

---

## 第一步：访问飞书开放平台

1. 打开浏览器，访问：https://open.feishu.cn/
2. 登录你的飞书账号
3. 进入「开发者后台」

---

## 第二步：创建 DocAgent 应用

### 2.1 创建应用
1. 点击「创建应用」
2. 选择「企业自建应用」
3. 填写应用信息：
   - **应用名称**：DocAgent
   - **应用描述**：文档知识域Agent，负责文档读写、Wiki编辑、知识库管理
   - **应用图标**：可以选择一个文档相关的图标
4. 点击「确定创建」

### 2.2 获取凭证
创建成功后，在「凭证与基础信息」页面可以看到：
- **App ID**（App ID）
- **App Secret**（App Secret）

**重要**：请记录下来，后面配置需要用到！

### 2.3 配置权限

在「权限管理」页面，添加以下权限：

| 权限名称 | 权限描述 | 是否必须 |
|---------|---------|---------|
| `docx:document` | 查看、编辑、删除飞书文档 | 是 |
| `wiki:wiki` | 查看、编辑、删除知识库 | 是 |
| `drive:drive` | 查看、编辑、删除云空间文件 | 是 |

### 2.4 发布版本（可选，测试时可以先不用）

---

## 第三步：创建 CalendarAgent 应用

### 3.1 创建应用
1. 点击「创建应用」
2. 选择「企业自建应用」
3. 填写应用信息：
   - **应用名称**：CalendarAgent
   - **应用描述**：日程协调域Agent，负责日程管理、会议安排、提醒通知
   - **应用图标**：可以选择一个日历相关的图标
4. 点击「确定创建」

### 3.2 获取凭证
同样在「凭证与基础信息」页面记录：
- **App ID**
- **App Secret**

### 3.3 配置权限

在「权限管理」页面，添加以下权限：

| 权限名称 | 权限描述 | 是否必须 |
|---------|---------|---------|
| `calendar:calendar` | 查看、编辑、删除日历 | 是 |
| `calendar:event` | 查看、编辑、删除日程 | 是 |
